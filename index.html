Python for gamma-ray astronomy

A workshop at MPIK Heidelberg, November 2 - 6, 2015

About Sessions Join Participants

About

We are holding a week-long (November 2 - 6, 2015) workshop on Python for gamma-ray astronomy at MPIK Heidelberg.

A key aspect is to bring the Sherpa, Gammapy and Naima Python package developers together for a coding sprint, to build high-level science analysis tools for gamma-ray data from H.E.S.S., CTA and Fermi.

A second goal of the workshop is to grow the user and contributor base and improve the coding and analysis skills of all participants. To that end, a number of presentations and tutorials will be scheduled.

We'd also love to have a more general and diverse workshop that includes developers working on similar codes (e.g. Gammalib/ctools or 3ML) or other gamma-ray telescopes (e.g. VERITAS, MAGIC or HAWC), to exchange ideas, improve interoperability, cross-check and foster more collaboration.

Sessions

The format of this workshop will be modeled roughly on the Python in Astronomy workshop at the Lorentz center in April 2015, i.e. there will be some pre-scheduled presentations, tutorials and discussion sessions, but most of the time is left free for participants to discuss and work on what they like in small groups (a so-called unconference).

Please note that the following agenda is very preliminary.
If you'd like to come and present something, please send an email to Christoph Deil.

We'll start with plenary presentations / demos / discussions on Monday:

  1. Sherpa GH #76 -- Python 3 support
  2. Sherpa Sphinx API docs (including the object-oriented API)
  3. Sherpa GH #73 -- Poisson statistics with background - implementing wstat
The rest of the week (Tuesday to Friday) will be mostly a free-form work meeting. Workshop participants propose things (features, checks, analyses) they'd like to discuss and work on and from small sub-groups to work on in parallel. There will be a few hands-on tutorials. Major features / issues:
  1. Sherpa GH #76 -- Python 3 support
  2. Sherpa Sphinx API docs (including the object-oriented API)
  3. Sherpa GH #73 -- Poisson statistics with background - implementing wstat
Minor features / issues:
  1. Sherpa GH #43 -- Generalize the image PSF to allow data sub-pixel binning

Join

Anyone is welcome to join the workshop. There is no registration fee.
Just send an email to Christoph Deil to register, or let me know if you have any questions.

Most participants are expected to pay for travel and hotel themselves. There is the option to stay at the MPIK guest house (TODO: cost?) Please contact Peter Eger if you'd like to book a room at the MPIK guest house or if you have any questions concerning accomodation.

If you're new to Python, we recommend that you spend some time to learn it before the workshop in the context of scientific computing and Astronomy.

  • The Python Scientific Lecture Notes are a good tutorial to learn about Numpy, Scipy and Matplotlib.
  • The Practical Python for Astronomers tutorials are a good resource to learn about Astropy and specifically the Fitting and Modeling 1-D and 2-D Data tutorial to get started with Sherpa.
  • Since Python is very popular overall, and specifically in Astronomy, there's of course a ton of other free tutorials, books and resources available online, e.g. the Python for Astronomers web site or the Astropy tutorials
  • If you don't have a scientific Python stack installed on your computer, try it out online at https://try.jupyter.org/ (this uses the Jupyter notbook, formerly known as the IPython notebook) and then to install it on your machine, we recommend Anaconda as a free, cross-platform (Linux, Mac, Windows) scientific Python distribution.

If you're thinking about contributing to one of the open-source projects discussed at this workshop, you should learn a bit about the git version control system and https://help.github.com/categories/bootcamp/ http://software-carpentry.org/
