About

We are holding a week-long (November 2 - 6, 2015) workshop on Python for gamma-ray astronomy at MPIK Heidelberg.

A key aspect is to bring the Sherpa, Gammapy and Naima Python package developers together for a coding sprint, to build high-level science analysis tools for gamma-ray data from H.E.S.S., CTA and Fermi.

A second goal of the workshop is to grow the user and contributor base and improve the coding and analysis skills of all participants. To that end, a number of presentations and tutorials will be scheduled.

We'd also love to have a more general and diverse workshop that includes developers working on similar codes (e.g. Gammalib/ctools or 3ML) or other gamma-ray telescopes (e.g. VERITAS, MAGIC or HAWC), to exchange ideas, improve interoperability, cross-check and foster more collaboration.

Content

The format of this workshop will be modeled roughly on the Python in Astronomy at the Lorentz center in April 2015, i.e. a combination of pre-scheduled and free time (Unconference") We'll start with plenary presentations or demos and discussions on Monday. The rest of the week (Tuesday to Friday) will be mostly a free-form work meeting. Workshop participants propose things (features, checks, analyses) they'd like to discuss and work on and from small sub-groups to work on in parallel. There will be a few hands-on tutorials. Major features / issues:

  1. Sherpa GH #76 -- Python 3 support
  2. Sherpa Sphinx API docs (including the object-oriented API)
  3. Sherpa GH #73 -- Poisson statistics with background - implementing wstat
Minor features / issues:
  1. Sherpa GH #43 -- Generalize the image PSF to allow data sub-pixel binning
